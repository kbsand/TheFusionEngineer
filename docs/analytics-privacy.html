이용 통계 안내

재방문자를 식별하지 않는 세션 단위 이용 통계

게임 개선과 포트폴리오의 이용 현황 확인을 위해 게임 실행별 통계를 수집합니다. 새로고침하거나 다시 접속하면 이전 실행과 연결되지 않는 별도 세션으로 집계됩니다.

수집하는 항목

  • 현재 페이지 실행 중 메모리에만 유지되는 무작위 세션 ID
  • 활성 플레이 시간
  • 도달한 최고 스테이지와 엔딩 도달 여부
  • 게임 빌드 버전
  • 서버가 기록하는 세션 시작 시각과 마지막 갱신 시각

수집하지 않는 항목

이 이용 통계 기능은 이름, 이메일, 전화번호, GitHub 계정, IP 주소, 위치정보, 브라우저·운영체제·화면·기기정보, 언어, 방문 경로, 검색어 또는 광고 식별자를 수집하거나 애플리케이션 데이터베이스에 저장하지 않습니다. 재방문을 연결하거나 개인별 프로파일을 만들지 않습니다.

이용 통계 기능은 Cookie, LocalStorage, SessionStorage, IndexedDB, Unity PlayerPrefs에 세션 ID나 통계 상태를 기록하지 않습니다. Unity WebGL 플레이어는 게임 파일을 빠르게 다시 불러오기 위한 브라우저 캐시 기능을 별도로 사용할 수 있으며, 이 캐시는 이용 통계나 재방문 식별에 사용되지 않습니다.

보관과 이용

개별 세션 원본과 세션 ID가 없는 날짜별 합계는 자동으로 삭제하지 않습니다. 저장된 통계는 운영자가 접근권한을 제한해 관리합니다. 통계는 게임 품질과 스토리 진행 흐름을 개선하는 목적으로만 사용하며, 광고, 개인별 평가·프로파일링, 제3자 판매에 사용하지 않습니다.

Cloudflare 사용

통계 전송과 저장에는 Cloudflare Worker와 D1을 사용합니다. Cloudflare 인프라는 네트워크 요청을 제공하는 과정에서 자체 정책에 따라 기술 정보를 처리할 수 있지만, 이 애플리케이션의 Worker 코드는 IP 주소나 기기 정보를 읽어 D1에 저장하지 않습니다.

검토 안내

이 안내는 구현과 운영 검토를 돕기 위한 설명이며 법률 자문이나 특정 법률에 대한 적합성 보장이 아닙니다. 운영자는 공개 전과 운영 중에 목적, 수집 범위, 보관 기간을 정기적으로 다시 검토해야 합니다.
